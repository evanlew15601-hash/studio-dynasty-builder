# Studio Magnate — Help

## Quick start

- Quick Start starts a default studio and immediately begins the game.
- Custom Studio lets you set your studio name, choose up to 3 specialty genres, and pick a difficulty.

## Saving & loading

- Use the in-game Save option to save your current run.
- From the main menu, use Load Saved Game to continue.
- Desktop builds store saves as files on your computer (see the in-game Saves… dialog to view the exact folder).



## Difficulty

- Easy: more starting budget and a more forgiving market.
- Normal: balanced experience.
- Hard: tighter budget and more competition.
- Magnate: the toughest start.

## Tips

- Specialize early: aligning your projects with your specialties helps you build momentum.
- Cash flow matters: keep enough budget buffer for marketing, talent, and overruns.
- Talent relationships compound over time—invest in people you want to keep.

## Mods (if enabled)

If you are using mod bundles, keep a backup of your save before changing mods. Some changes can affect existing saves.

## Troubleshooting

- If you see odd behavior after updates, try starting a new run to confirm it isn't save-data related.
- If the UI looks broken, reload the app (or restart the desktop app).
